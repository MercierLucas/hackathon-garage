Outils

Si vous êtes nouveau et que vous ne savez pas quoi utiliser comme outils nous vous conseillons d'installer la suite Anaconda. Celle-ci contient toutes les dépendances majeures dont vous aurez besoin ( du moins au début) lors de ce challenge. La suite vient avec plusieurs outils dont Jupiter notebook.

Pour l'installer rien de plus simple il vous suffit de vous rendre sur la page de téléchargement d'anaconda (vous aurez peut être besoin de redémarrer votre ordinateur)

Si vous n'avez pas python installé, choisissez la version 3.7 (la plus récente) pour un meilleur support des dépendances

Il est également possible d'utiliser l'alternative proposé par Google: Google Colab, un outil collaboratif basé sur les notebooks.
